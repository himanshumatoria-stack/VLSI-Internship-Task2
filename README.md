# VLSI Task 2 - Verilog HDL
This repository contains the Verilog HDL implementation and simulation of basic digital circuits.

## Implemented Circuits
- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- Half Adder
- Full Adder

## Tools Used
- Verilog HDL
- ModelSim Intel FPGA Starter Edition 2020.1

## Repository Contents
Each circuit includes:
- Verilog design (.v)
- Testbench (.v)
